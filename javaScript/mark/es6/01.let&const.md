### let与const

#### 作用

let 用于声明变量

const 用于声明常量



#### 特点

1. let与const 声明变量会在外层形成一个块级作用域，作用域外无法获取到变量
2. 无法状态提升，在变量声明之前引用会报错
3. **const** 声明一个只读的常量，一旦声明，常量的值就不能改变
4. 不能重复声明



**暂时性死区**： 

```
var tmp = 123;

if (true) {
  tmp = 'abc'; // ReferenceError
  let tmp;
}
```

像这种在if中有用let声明的会生成一个块级作用域，在这里面在声明之前使用的变量就会报错，叫暂时性死区
